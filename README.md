# Drill4

This project is a fork of [Drill3pro](https://github.com/kacperwyczawski/drill3pro).

## Sucessor to [Drill3](https://github.com/Matb85/drill3)

Drill4 is a web application designed to help users create and take quizzes based on custom question sets. Users can upload question files, configure test settings, and receive detailed results upon completion.

### Drilluj to jak małpa
- Spaced repetiotion - dzięki algorytmowi z Anki/Quizleta nie musisz już nic myśleć. Poprostu klikasz do skutku i jak zrobisz coś źle - to to pytanie bedzie wracać aż sie go nie nauczysz.
- Osobno tryb testu z liczeniem punktow i osobno tryb learn ze spaced repetition
- Automatyczne rozpoznawanie czy drill jest jednokrotnego czy wielokrotnego wyboru - dzięki temu przy drillach jednokrotnego wyboru masz mniej klikania i lecisz z tym szybciej
- Obsluga klawiatury zeby leciec z tym szybciej
