# Drill3Pro

This project is a fork of [Drill3](https://github.com/Matb85/drill3).
## Sucessor to [Drill2](https://github.com/gronostajo/drill2)

Drill3Pro is a web application designed to help users create and take quizzes based on custom question sets. Users can upload question files, configure test settings, and receive detailed results upon completion.
